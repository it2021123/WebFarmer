Τεχνολογίες που χρησιμοποιήθηκαν στην ανάπτυξη του κώδικα




Spring Boot :Παρέχει ένα ενσωματωμένο περιβάλλον ρύθμισης και προκαθορισμένες ρυθμίσεις που μας επέτρεψε να ασχοληθούμε κυρίως με τη λογική της εφαρμογής. Ο ενσωματωμένος web server του εξαλείφει την ανάγκη για εξωτερικούς servers, ενώ η διαχείριση και η διαμόρφωση εξαρτήσεων είναι αυτόματες.


Spring Security: Παρέχει λειτουργίες πιστοποίησης, εξουσιοδότησης και προστασίας των πόρων, επιτρέποντας να προσθέσουμε εύκολα και αποτελεσματικά μηχανισμούς ασφαλείας στις εφαρμογές .


Spring Data JPA: Βοηθά στη διαχείριση δεδομένων σε βάσεις δεδομένων χρησιμοποιώντας το Java Persistence API (JPA). Βασικά χρησιμοποιείται για την απλοποίηση των λειτουργιών που αφορούν την αποθήκευση, την ανάκτηση, την ενημέρωση και τη διαγραφή δεδομένων σε μια βάση δεδομένων.


Βάση Δεδομένων : Postgresql
